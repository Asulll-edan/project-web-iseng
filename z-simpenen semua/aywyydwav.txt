==================================================
CHECKOUT & ORDER FLOW
==================================================

Flow order:

1. User memilih menu
2. Tambah ke cart
3. Checkout
4. Generate nomor order random otomatis
5. Status awal = menunggu
6. Pesanan masuk realtime ke halaman kasir
7. Kasir melihat detail pesanan
8. Kasir mengubah status:
   - menunggu
   - cooking
   - selesai
9. Jika status selesai:
   - user mendapatkan tombol “Selesaikan Pesanan”
10. Jika user menekan selesai:
   - order benar-benar completed
   - loyalty point bertambah
   - cashback membership masuk otomatis
